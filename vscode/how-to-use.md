# How to Use

```bash
code --list-extensions | xargs -L 1 echo code --install-extension
```

## Active Key Bindings

Press `Ctrl + Shift + P`, type `json shortcuts` and look for `Preferences: Open Keyboard Shortcuts (JSON)`.

Update the current file with `keybindings.json` on this repository.

## Active Settings

Press `Ctrl + Shift + P`, type `json settings` and look for `Preferences: Open Settings (JSON)`.

Update the current file with `settings.json` on this repository.
